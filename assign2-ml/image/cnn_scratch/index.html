CNN Scratch

Baseline end-to-end image classifier trained from scratch on the preprocessed RSITMD image branch. This page highlights the pipeline from input to output and the key building blocks in the script.

Pipeline
Input
RSITMD image crops
Source data is multimodal RSITMD, but this model uses the image branch only.
Preprocess
Resize + crop + normalize
RandomResizedCrop, flip, jitter, then tensor normalization to 224×224.
Backbone
ResNet18 from scratch
No ImageNet weights; the full network is initialized with weights=None.
Head
Dropout + Linear classifier
Final FC layer maps features to all RSITMD classes.
Output
Class logits + metrics
Accuracy, balanced accuracy, macro F1, confusion matrix, and learning curves.
What makes it different
Scratch baseline
A pure baseline for comparison against pretrained and attention-based variants.
