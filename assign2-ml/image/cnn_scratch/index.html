ResNet18-Scratch

Baseline end-to-end image classifier trained from scratch on the preprocessed RSITMD image branch. This page highlights the pipeline from input to output and the key building blocks in the script.

Pipeline
Input
RSITMD image crops
Source data is multimodal RSITMD, but this model uses the image branch only.
Preprocess
Augmentation + Tensors
Applies basic transforms (Resize 224×224, light crops, and horizontal flips) to provide minimal regularization while preparing tensors for the model.
Backbone
ResNet18 (weights=None)
A standard ResNet18 architecture where every layer is initialized with random values, requiring the model to learn all features (edges, textures) from the RSITMD data alone.
Head
Linear Classifier
A single linear layer that maps the 512 backbone features to the 33 target classes, optimized using Cross Entropy Loss.
Output
Class logits + metrics
Accuracy, balanced accuracy, macro F1, confusion matrix, and learning curves.
What makes it different
Random Weight Initialization
Utilizes the ResNet18 architecture but is trained entirely from scratch with random weight initialization (no ImageNet). This serves as the fundamental baseline to measure the impact of pre-training and more complex attention-based models on the RSITMD dataset.
