ResNet50-Frozen

A compact ResNet50-based image classifier built as a clean baseline. The script emphasizes a straightforward preprocess → backbone → classifier path, plus standard evaluation outputs.

Pipeline
Input
RSITMD image crops
Images are drawn from the source multimodal dataset and used for the image classification branch.
Preprocess
Augmentation + Tensors
Images are resized to 224×224 and normalized using ImageNet statistics. The pipeline includes light random cropping and horizontal flips during training to improve generalizability.
Backbone
ResNet50 encoder
The script uses a pretrained ResNet50 feature extractor.
Head
Linear Classifier
The FC layer is replaced for RSITMD class prediction.
Output
Validation predictions
Confusion matrix and example-level inspection artifacts.
What makes it different
Simple strong baseline
Good for comparing against the scratch model and the attention-based MBLANet variant.
