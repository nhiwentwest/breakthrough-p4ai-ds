ResNet50-SVM

A two-stage classifier: first extract image embeddings using a pretrained ResNet50 backbone, then train an SVM on the fixed feature vectors.

Pipeline
Input
RSITMD image crops
The image branch is processed as a classification dataset, not a captioning task.
Preprocess
Augmentation + Tensors
Images are resized to 224×224 and normalized. Features are extracted once and cached for SVM training.
Backbone
ResNet50 (Fixed)
Acts as a frozen feature extractor using ImageNet-1K V1 weights. The final fully connected layer is replaced with an Identity mapping to produce raw 2048-dimensional embeddings.
Classifier
RBF SVM
A classical Support Vector Machine with a Radial Basis Function kernel is trained on the extracted features. This separates representation learning from classification.
Output
Class predictions
Accuracy, balanced accuracy, macro F1, and confusion matrix on the RSITMD test split.
What makes it different
Hybrid classical pipeline
This model separates deep feature extraction from classical ML classification, making it useful as a strong non-end-to-end baseline.
