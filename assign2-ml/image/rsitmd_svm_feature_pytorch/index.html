Assignment 02 · Model Child Page

SVM + ResNet50 Features

A two-stage classifier: first extract image embeddings using a pretrained ResNet50 backbone, then train an SVM on the fixed feature vectors. The page follows the same visual pattern as the MBLANet child page, with the pipeline on top and separate result blocks for non-augmentation and augmentation.

Pipeline
Input
RSITMD image crops
The image branch is processed as a classification dataset, not a captioning task.
Preprocess
Resize + normalize + cache
Images are transformed to 224×224 tensors before feature extraction.
Backbone
ResNet50 extractor
The FC head is replaced with Identity to produce dense features for every image.
Classifier
RBF SVM
A classical classifier consumes the fixed embeddings from the extractor and produces class predictions.
Output
Test predictions + report
Reports accuracy, balanced accuracy, macro F1, test confusion matrix, learning curves, and saved artifacts.
What makes it different
Hybrid classical pipeline
This model separates deep feature extraction from classical ML classification, making it useful as a strong non-end-to-end baseline.
