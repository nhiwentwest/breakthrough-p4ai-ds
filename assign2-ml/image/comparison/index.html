Evaluation Method
Multi-Class Performance Metrics (C = 33)

All models are compared using three primary metrics, selected to reflect both per-class fairness and overall classification quality: Macro-Averaged F1 Score, Balanced Accuracy (macro recall), and Overall Accuracy.

ConditionInterpretation
True Positive (TP)Samples belonging to Class i that are correctly predicted as Class i.
True Negative (TN)Samples not belonging to Class i that are correctly predicted as any other class.
False Positive (FP)Samples not belonging to Class i but are incorrectly predicted as Class i.
False Negative (FN)Samples belonging to Class i but are incorrectly predicted as a different class.
1. Base Metrics (Per Class i)
Pi = TPi / (TPi + FPi)
Ri = TPi / (TPi + FNi)
F1i = 2 × (Pi × Ri) / (Pi + Ri)
2. Final Evaluation Metrics
OverallAcc = Σ TPi / N
BalancedAcc = (1/C) ΣCi=1 Ri
MacroF1 = (1/C) ΣCi=1 F1i

Key Insights
🏆
Best Overall: MBLANet
Achieves the highest test accuracy (0.9401) and macro F1 (0.9130) under light augmentation, demonstrating the power of attention mechanisms.
⚡
Fastest Training: ResNet50-SVM
Completes training in under 1 minute by separating feature extraction from classification, at the cost of lower accuracy.
📉
Heavy Aug ≠ Better
Most models show degraded test performance with heavy augmentation, suggesting over-regularization on this small dataset (4743 images).
🎯
Fine-Tune > Freeze
ResNet50-FineTune outperforms ResNet50-Frozen by ~8% F1, confirming that end-to-end adaptation is critical for domain-specific tasks.
