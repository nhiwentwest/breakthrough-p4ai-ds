MBLANet

A paper-aligned ResNet50 variant with CCAM + LSAM attention blocks injected into the backbone. This page emphasizes the attention pipeline and where the local/global branches act on the feature maps.

Pipeline
Input
RSITMD image crops
Preprocessed image branch from the multimodal source dataset.
Preprocess
Augment + normalize
Train-time crop/flip/jitter, eval-time resize to 224×224.
Backbone
ResNet50 + CLAM blocks
CCAM and LSAM are injected into layers 1–4 for local/global attention.
Head
Linear classifier
Final fully connected layer maps the backbone features to class logits.
Output
Attention-aware logits
Supported by Grad-CAM, saliency, and internal attention visualizations.
What makes it different
Attention baseline
The most architecture-heavy image model in the assignment, combining pretrained features and custom local attention modules.
