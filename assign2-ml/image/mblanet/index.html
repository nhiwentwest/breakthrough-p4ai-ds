MBLANet

Multi-branch lightweight attention network for RSITMD image classification. This page follows the same layout pattern as the fine-tune page: pipeline and source information on top, then each result table followed by its two images.

Pipeline
Input
RSITMD image crops
Image-only branch used for evaluation on the test split.
Preprocess
Resize + normalize
Standard image transform pipeline compatible with the model backbone.
Backbone
ResNet50 + MBLANet attention blocks
A pretrained ResNet50 is used as the feature extractor, then each residual stage is upgraded with CLAM modules so the network can focus on both global cues and local discriminative regions.
Attention
CCAM + LSAM inside CLAM
CCAM learns channel importance from pooled feature summaries, while LSAM builds a spatial attention map from local patches. Their outputs are multiplied with the feature map before the residual skip connection is added back.
Head
Classification head
A final linear layer produces logits over all RSITMD classes. Training tracks macro F1 on validation data for checkpointing and early stopping.
Output
Predictions + metrics
Reports test accuracy, balanced accuracy, macro F1, confusion matrix, learning curves, and inference speed / GPU peak memory.
What makes it different
Attention-based image model
A compact alternative to ResNet-style baselines, useful for comparing representation quality and training behavior.
