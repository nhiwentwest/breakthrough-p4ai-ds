RSITMD Preprocess

This page documents the RSITMD preprocessing pipeline used to build the cleaned 256×256 dataset. It summarizes the script’s steps, the processed dataset splits, and a compact snapshot of the dataset structure after preprocessing.

Pipeline
Input
Raw RSITMD dataset
The original multimodal RSITMD source is loaded from disk, then cleaned and converted into a structured Hugging Face dataset.
Cleaning
Caption normalization + duplicate filtering
Captions are lowercased, whitespace-normalized, tokenized, and scored to detect weak or noisy textual samples.
Image checks
Blank-image / pixel statistics
The script inspects images for empty samples and measures brightness, texture, and channel balance to support quality filtering.
Split
Train / validation / test
The final dataset is stratified into three splits and stored as a Hugging Face `DatasetDict`.
Output
processed_rsitmd_256_clean
A cleaned 256×256 image dataset with label metadata, filenames, and captions ready for downstream image and multimodal tasks.
