ResNet50-FineTune

A ResNet50 ImageNet backbone fine-tuned end-to-end on the preprocessed RSITMD image branch. This page shows the full pipeline and the role of the pretrained backbone.

Pipeline
Input
RSITMD image crops
Image branch only, treated as an image classification dataset.
Preprocess
Augmentation + Tensors
Images are resized to 224×224 and normalized using ImageNet statistics. During training, light random crops and flips are applied to enhance the model's robustness to orientation and position.
Backbone
ResNet50 (Pre-trained)
Uses the ResNet50_Weights.IMAGENET1K_V2 weights. Unlike frozen configurations, all backbone parameters are unlocked (requires_grad=True) to allow end-to-end fine-tuning on the RSITMD domain.
Head
Linear Classifier
The final 2048-feature layer is replaced with a single linear layer mapping to 33 classes. This allows the model to adapt its high-level decision boundary to the specific categories of the remote sensing dataset.
Output
Fine-tuned logits
Validation confusion matrix and macro-F1 driven checkpointing.
What makes it different
End-to-End Fine-Tuning
Leverages a pretrained ResNet50 backbone with high-quality ImageNet-1K V2 weights, adapting the entire network's parameters to the target domain. This configuration typically yields the best balance of speed and accuracy compared to training from scratch.
