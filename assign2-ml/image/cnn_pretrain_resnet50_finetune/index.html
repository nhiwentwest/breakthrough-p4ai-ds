Pretrained CNN Fine-tune

A ResNet50 ImageNet backbone fine-tuned end-to-end on the preprocessed RSITMD image branch. This page shows the full pipeline and the role of the pretrained backbone.

Pipeline
Input
RSITMD image crops
Image branch only, treated as an image classification dataset.
Preprocess
Resize + normalize
224×224 transform stack matching ImageNet-style training.
Backbone
ResNet50 pretrained
ImageNet weights loaded and all layers trainable.
Head
Dropout + Linear classifier
Final classification head replaces the ImageNet FC layer.
Output
Fine-tuned logits
Validation confusion matrix and macro-F1 driven checkpointing.
What makes it different
Transfer learning
Best for comparing against the scratch baseline and measuring the benefit of ImageNet initialization.
