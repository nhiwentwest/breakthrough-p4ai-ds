Random Forest

Bagging-based regression model from ml_tabular_insurance1.ipynb.

Pipeline

  • Input: insurance rows with numeric and categorical fields
  • Preprocess: one-hot encoding + standard scaling
  • Head: RandomForestRegressor(n_estimators=100, random_state=42)
  • Evaluation: RMSE, MAE, and R²
  • Analysis: residual plots and comparison against the linear baseline

Key idea

Uses multiple trees and bagging to capture non-linear relationships in the insurance data.
