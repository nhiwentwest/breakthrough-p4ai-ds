Regression Model Suite

Evaluation of three fundamental regression algorithms—Linear Regression, Random Forest, and Gradient Boosting—on the insurance dataset to predict medical charges. This page analyzes the accuracy and performance trade-offs across the suite.

Preprocessing & Strategy
Missing Values
Verified 0% missing data
All 1,338 samples are complete, so no imputation was required.
Data Splitting
80% Training / 20% Testing
1,070 training samples and 268 testing samples; random_state=42 was fixed for reproducibility.
Feature Engineering
One-Hot Encoding + Standardization
Categorical variables (sex, smoker, region) were converted to numeric binary columns, while age and bmi were scaled with StandardScaler to a common range.
Pipeline Heads
Input → Backbone → Heads → Output
The pipeline loads raw CSV data into a Pandas DataFrame, fits the scaler only on the training set to prevent leakage, then passes the scaled features into Linear Regression, Random Forest, and Gradient Boosting heads for continuous insurance charge prediction.
