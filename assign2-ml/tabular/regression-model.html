Regression Model Suite

Evaluation of three regression algorithms—Linear Regression, Random Forest, and Gradient Boosting—on the Insurance dataset (1,338 records, 8 features after OHE). The target charges exhibits a bimodal distribution driven by the smoker/non-smoker split, which challenges purely linear models.

Pipeline
Input
Raw CSV → Pandas DataFrame
Loading raw insurance data into a structured tabular format for processing.
Preprocess
Missing values, encoding, scaling
Verified 0% missing data (1,338/1,338 samples are complete). Categorical variables (sex, smoker, region) are converted with One-Hot Encoding, while age and bmi are standardized with StandardScaler.
Backbone
Scaler fit on training only
StandardScaler is fit only on the training set to prevent data leakage. Split: 70/15/15 → 936 train / 201 valid / 201 test, random_state=42.
Head
Linear Regression / RF / GBDT
Scaled features are passed into three model heads to predict continuous insurance charges. Linear Regression is the baseline, Random Forest averages 100 trees, and Gradient Boosting sequentially corrects errors from previous trees.
Output
Predicted Charges
Continuous numerical prediction of insurance charges for unseen customers.
What makes it different
End-to-End Pipeline
This notebook implements a complete regression research flow. It automates the transition from raw data to insights through a multi-stage pipeline: robust categorical encoding, standardized scaling, multi-model benchmarking (LR, RF, GBDT), and rigorous evaluation via residual plots and learning curves.

Linear Regression · Predict vs Actual

Predicted vs actual charges for Linear Regression. Systematic under-prediction visible for high-cost smoker patients due to the missing BMI×smoker interaction term.

Linear Regression · Residual Plot

LR residuals show a funnel-shape (heteroscedasticity) — increasing spread with larger predictions, confirming systematic failure on the smoker sub-population.

Linear Regression · Learning Curves

Train and validation curves converge quickly to a relatively high error — classic underfitting (high bias). The model is too simple for the non-linear data structure.

Random Forest · Predict vs Actual

RF predictions cluster more tightly around the diagonal. The 100-tree ensemble captures the smoker interaction automatically via branching.

Random Forest · Residual Plot

RF residuals are more uniformly distributed around zero, though some variance remains at extreme values due to bootstrap sampling noise.

Random Forest · Learning Curves

Near-perfect training score with a moderate train-validation gap — mild overfitting that decreases with more data. Characteristic of high-variance ensemble models.

Gradient Boosting · Predict vs Actual

GB produces the tightest clustering around the diagonal. Sequential error correction yields the most accurate predictions across both smoker and non-smoker sub-populations.

Gradient Boosting · Residual Plot

GB residuals are more uniformly distributed around zero, confirming better model fit for the non-linear smoker–BMI interactions.

Gradient Boosting · Learning Curves

Validation curve closely tracks the training curve — stable generalisation with minimal overfitting thanks to shallow trees (max_depth=3).

Feature Importance (Gradient Boosting)

smoker_yes accounts for ≈67% of total predictive power, confirming the EDA finding. BMI (~19%) and age (~12%) together contribute another ~30%, while gender and region have minimal impact.
