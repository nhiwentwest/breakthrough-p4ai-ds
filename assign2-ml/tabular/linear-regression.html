Linear Regression

Baseline regression model from ml_tabular_insurance1.ipynb.

Pipeline

  • One-hot encode categorical columns
  • Standardize numerical features with StandardScaler
  • Fit LinearRegression() on the training split
  • Evaluate with RMSE, MAE, and R²
  • Inspect actual-vs-predicted scatter plot and residual plot

Key idea

Serves as a lightweight baseline to compare against non-linear ensemble methods.
