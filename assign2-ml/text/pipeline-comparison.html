Pipeline Comparison

Comparison framework from text/Pipeline_Comparision.py and Pipeline_Comparision.ipynb.

Pipeline

  • Input: the same financial news sentiment text
  • Preprocess: shared cleaning and duplicate / missing-value handling
  • Head: TF-IDF, dimensionality reduction, and classifier variants
  • Evaluation: accuracy, precision, recall, F1, confusion matrix
  • Analysis: compare classical pipelines side by side

Key idea

Provides the comparison layer between feature extractors, reducers, and classifiers under a unified setup.
