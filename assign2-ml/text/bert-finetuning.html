BERT Fine-tuning

Transformer-based sentiment pipeline from text/BERT-finetuning.py and BERT-finetuning.ipynb.

Pipeline

  • Input: raw tweet text
  • Preprocess: tokenizer with max length, padding, truncation
  • Backbone: BERT-base uncased encoder
  • Head: mean pooling + linear classifier
  • Evaluation: accuracy, precision, recall, F1, inference time

Key idea

Uses contextual embeddings to model sentiment directly from the token sequence.
