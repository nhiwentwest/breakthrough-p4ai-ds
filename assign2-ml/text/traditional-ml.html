Traditional ML

Classical sentiment pipeline from text/Traditional_ML.py and Traditional_ML.ipynb.

Pipeline

  • Input: Twitter financial news text
  • Preprocess: remove URLs, handle tickers, lowercase, clean punctuation, remove stopwords, drop duplicates and missing values
  • Vectorizer: TF-IDF with unigrams and bigrams
  • Head: classical classifier (traditional ML baseline)
  • Evaluation: accuracy, precision, recall, F1, confusion matrix

Key idea

A lightweight baseline that converts text into sparse vectors before classification.
