Analysis steps
📋 Step 1 — Dataset Overview Overview
Load dữ liệu, in shape, column names, dtypes, descriptions. Kiểm tra 5 dòng đầu. Feature engineering: bin GDP per capita thành GDP_Level (Low / Medium / High) dựa trên quantiles (33rd và 66th percentile).
Shape / dtypes GDP_Level engineering df.head()
✅ Step 2 — Missing Values Data Quality
Kiểm tra missing values trên toàn bộ dataset. Trực quan hóa bằng bar chart (horizontal). Tính tổng số và % missing mỗi column. Đưa ra imputation recommendations dựa trên feature type.
isnull().sum() Horizontal bar chart Imputation advice
📈 Step 3 — Numerical Distributions Univariate
Phân tích phân phối 7 numerical features: GDP per capita, Social support, Healthy life expectancy, Freedom to make life choices, Generosity, Perceptions of corruption, và Score. Mỗi feature có histogram + stats table (Mean, Median, Std, Min, Max).
Histograms Mean / Median / Std 7 features Selectable via UI
🏷️ Step 4 — Categorical: GDP Level Categorical
Feature engineering: binning GDP per capita thành 3 nhóm Low / Medium / High. Trực quan hóa bằng bar chart. Tính số lượng và tỉ lệ phần trăm country mỗi nhóm.
GDP_Level Low / Medium / High Bar chart Feature binning
🎯 Step 5 — Target Distribution Target
Phân tích phân phối của target variable — Happiness Score. Histogram với đường Mean và Median. Tính toán các thống kê tổng quát: mean, median, min, max.
Score distribution Histogram μ & median lines
🔗 Step 6 — Correlation Analysis Bivariate
Tính correlation matrix giữa 7 numerical features. Trực quan hóa bằng heatmap (RdBu_r colormap). Trích xuất các cặp feature có |r| > 0.5, nhận diện top predictors của Happiness Score.
Correlation heatmap RdBu_r colormap |r| > 0.5 threshold Feature pairs
📦 Step 7 — Outlier Detection (IQR) Outliers
Phát hiện outliers bằng IQR method: values outside [Q1 − 1.5·IQR, Q3 + 1.5·IQR]. Trực quan hóa bằng box plots song song trên 7 features. Tính Q1, Q3, IQR, lower/upper bounds, outlier count/percentage cho từng feature.
IQR method Box plots 7 features Outlier statistics
📊 Step 8 — GDP Level vs Score Grouped
So sánh Happiness Score giữa 3 nhóm GDP_Level (High / Medium / Low) bằng grouped box plot. Tính grouped statistics: mean, median, std, min, max theo từng nhóm. Nhận diện economic divide.
Grouped box plot GDP_Level × Score Economic divide Grouped stats
🔍 Step 9 — Sample Data Sample
Quan sát dữ liệu thô: top 10 happiest countries, bottom 10, và random sample (seed=42). Giúp hiểu trực quan về raw data trước khi đi vào kết luận.
