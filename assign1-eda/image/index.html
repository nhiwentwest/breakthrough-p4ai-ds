Analysis Steps
🗂️ Step 1 — Dataset Overview Overview
Kiểm tra tổng quan dataset: số lượng train/test images và labels, xác nhận shape đúng (60k train · 10k test). Reshape vector 784-D thành 28×28 matrix để visualize.
MNISTLoader 60k train / 10k test 784 → 28×28 reshape
✅ Step 2 — Data Validation Validation
Xác nhận dữ liệu đúng format bằng assertions: len(images) == 60000, len(test_images) == 10000. Kiểm tra pixel range: giá trị min/max (0–255).
assert len() Pixel range check Data integrity
📊 Step 3 — Sample Images Sample
Hiển thị grid 3×3 (9 ảnh đầu tiên) từ training set với cmap="gray". Mỗi ảnh hiển thị label tương ứng phía trên. Giúp quan sát trực quan dữ liệu thô.
matplotlib cmap="gray" 9 sample images
📈 Step 4 — Digit Distribution Univariate
Vẽ count plot phân bố số lượng mẫu mỗi digit (0–9) trong training set. Dùng seaborn.countplot. Kiểm tra class imbalance — MNIST thường tương đối balanced.
seaborn.countplot Class distribution Class imbalance check
🔢 Step 5 — Per-Digit Examples Sample
Hiển thị 1 ảnh đại diện cho mỗi digit (0–9), tìm first occurrence của mỗi class trong training set. Grid 2×5. Giúp xác nhận mỗi class có dữ liệu hợp lệ.
First occurrence per digit 2×5 grid np.where()
🎯 Step 6 — Mean Image per Digit Mean
Tính mean image cho mỗi digit bằng cách lấy trung bình pixel-wise trên tất cả ảnh cùng class. Reshape thành 28×28 và visualize bằng cmap="inferno". Cho thấy cấu trúc pixel đặc trưng mỗi chữ số.
Mean image per class cmap="inferno" Pixel-wise average 2×5 grid
